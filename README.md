## 初回セットアップ - Getting Started

node 22 latest (LTS)
yarn 4.4.1 (Corepack)

Corepack有効化
Enable corepack

```shell
corepack enable npm yarn pnpm
```

package.json に記述されている依存関係のインストール
Install dependencies listed in package.json

```shell
yarn
```

## Next.jsを開発モードで立ち上げ - Start Next.js in Development Mode

```shell
yarn dev
```
